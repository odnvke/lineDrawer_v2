 # Программа рисует линии, как рисовать записывается в json

главная идея:
- есть конструкция которая вызывается несколько раз
- тип соединения точек(может быть любое количество) в этой конструкции
- функции которые возврощают координаты этих точек
- параметры внутри функции могут содержать вырожения включая time - время;  n - номер итерации(1 пункт); angle_step - цисло, формула: angle_step = 2pi / count(количество итераций)
- функции например fixed(x, y); circle(size, angle); sqaure(size, angle); ngon(size, angle, sedes); ... суперэлипс; кривая Ламэ.
- функции для суммы, умножения, морфинга 2 функций


### архетектура позволяет достаточно легко расширяться как горизонтально(на уровне, программы, типов соедениния точек, функции) так и вертикально так как программа достотачно лёгкая (переписывалась с прев вепсии - 3d)
